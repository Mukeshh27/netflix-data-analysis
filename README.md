# Netflix Data Analysis

## Project Overview

This project explores and analyzes Netflix movies and TV shows to identify trends in content distribution. The analysis focuses on understanding the types of content available on Netflix, the most common genres, and how Netflix content has grown over the years.

## Dataset

The dataset used in this project is the **Netflix Movies and TV Shows dataset** obtained from Kaggle.

Dataset file:
netflix_titles.csv

The dataset contains information such as:

* Show ID
* Title
* Type (Movie or TV Show)
* Director
* Cast
* Country
* Release Year
* Rating
* Duration
* Genre

## Tools and Technologies

This project was built using:

* Python
* Pandas
* Matplotlib

## Steps in the Analysis

1. Import the required Python libraries.
2. Load the Netflix dataset using Pandas.
3. Explore the dataset structure and check for missing values.
4. Analyze different attributes such as content type, country, and genre.
5. Create visualizations to understand content trends.

## Key Questions Explored

* What is the ratio of Movies vs TV Shows on Netflix?
* Which countries produce the most Netflix content?
* What are the most common genres available?
* How has Netflix content grown over time?

## Visualization

Charts were created using Matplotlib to visualize:

* Distribution of Movies vs TV Shows
* Top countries producing Netflix content
* Most popular genres

## Key Insights

* Netflix has significantly more Movies compared to TV Shows.
* Certain countries contribute a large portion of the available content.
* Some genres appear more frequently than others on the platform.

## How to Run the Project

1. Download the dataset from Kaggle.
2. Place the file `netflix_titles.csv` in the same folder as the Python script.
3. Run the Python script to perform the analysis and generate the visualizations.
